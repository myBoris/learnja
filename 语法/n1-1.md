# N1 笔记

## <span style="color:#d9485f;">1. ～あっての</span>

- 中文意思：正因为有……才有……；……是……成立的基础。
- 日语例句：お客様あっての商売だから、失礼のないようにしなければならない。
- 例句读音：おきゃくさまあってのしょうばいだから、しつれいのないようにしなければならない。
- 翻译：正因为有顾客，生意才能成立，所以必须注意不能失礼。

---

## <span style="color:#2b6cb0;">2. ～以外の何ものでもない</span>

- 读音：いがいのなにものでもない
- 中文意思：无非就是……；正是……；不外乎是……。
- 日语例句：毎日努力を続けられたのは、家族の支え以外の何ものでもない。
- 例句读音：まいにちどりょくをつづけられたのは、かぞくのささえいがいのなにものでもない。
- 翻译：能够每天坚持努力，这无非就是家人的支持所带来的结果。

---

## <span style="color:#2f855a;">3. ～いかんで/いかんでは/いかんによっては/いかんにかかっている/いかんだ</span>

- 中文意思：根据……而定；取决于……；要看……如何。
- 日语例句：
  1. 成績いかんで、進学できる大学が決まる。
  2. 対応いかんでは、大きな問題に発展することもある。
  3. 天候いかんによっては、試合は中止になるかもしれない。
  4. この計画が成功するかどうかは、皆の協力いかんにかかっている。
  5. 採用するかどうかは、面接の結果いかんだ。
- 例句读音：
  1. せいせきいかんで、しんがくできるだいがくがきまる。
  2. たいおういかんでは、おおきなもんだいにはってんすることもある。
  3. てんこういかんによっては、しあいはちゅうしになるかもしれない。
  4. このけいかくがせいこうするかどうかは、みなのきょうりょくいかんにかかっている。
  5. さいようするかどうかは、めんせつのけっかいかんだ。
- 翻译：
  1. 根据成绩，可以升学的大学会有所不同。
  2. 根据应对方式不同，也可能发展成大问题。
  3. 根据天气情况，比赛也许会取消。
  4. 这个计划能否成功，取决于大家的合作。
  5. 是否录用，要看面试结果。

## <span style="color:#b7791f;">4. ～いかんにかかわらず/いかんによらず/いかんをとわず</span>

- 中文意思：不论……如何；不管……怎样。
- 日语例句：
  1. 経験の有無いかんにかかわらず、やる気のある人を歓迎します。
  2. 理由のいかんによらず、締め切りを過ぎた提出は認められません。
  3. 国籍いかんをとわず、このイベントには誰でも参加できます。
- 例句读音：
  1. けいけんのうむいかんにかかわらず、やるきのあるひとをかんげいします。
  2. りゆうのいかんによらず、しめきりをすぎたていしゅつはみとめられません。
  3. こくせきいかんをとわず、このイベントにはだれでもさんかできます。
- 翻译：
  1. 无论有没有经验，都欢迎有干劲的人。
  2. 不论理由如何，超过截止日期提交的材料一律不予受理。
  3. 不问国籍，任何人都可以参加这个活动。

## <span style="color:#d9485f;">5. いざ～となると/いざ～となれば/いざ～となったら</span>

- 中文意思：一到关键时刻就……；真到了……的时候就……。
- 日语例句：
  1. 独立したいと思っていたが、いざ一人で店を出すとなると不安も大きい。
  2. 普段は冷静な彼でも、いざ本番となれば緊張するらしい。
  3. いつか海外で働きたいと思っていても、いざその話が現実となったら迷ってしまう人は多い。
- 例句读音：
  1. どくりつしたいとおもっていたが、いざひとりでみせをだすとなるとふあんもおおきい。
  2. ふだんはれいせいなかれでも、いざほんばんとなればきんちょうするらしい。
  3. いつかかいがいではたらきたいとおもっていても、いざそのはなしがげんじつとなったらまよってしまうひとはおおい。
- 翻译：
  1. 虽然一直想独立，但真到了要一个人开店的时候，不安也很大。
  2. 就算是平时很冷静的他，一到正式上场时似乎也会紧张。
  3. 虽然很多人都想着将来去国外工作，但真当这件事变成现实的时候，还是会犹豫。

## <span style="color:#2b6cb0;">6. 今ひとつ～(ない)</span>

- 读音：いまひとつ
- 中文意思：总觉得还差一点；另一个；还不够……。
- 日语例句：この説明は今ひとつ分かりにくいので、例を入れてほしい。
- 例句读音：このせつめいはいまひとつわかりにくいので、れいをいれてほしい。
- 翻译：这个说明总觉得还是不太好懂，希望能加个例子。

## <span style="color:#2f855a;">7. 言わずもがな</span>

- 读音：いわずもがな
- 中文意思：不用说也知道；不言而喻。
- 日语例句：健康にとって睡眠が大切なのは言わずもがなだ。
- 例句读音：けんこうにとってすいみんがたいせつなのはいわずもがなだ。
- 翻译：睡眠对健康很重要，这是不言而喻的。
- 和类似表达对比：
  
  | 表达 | 感觉 |
  | --- | --- |
  | 言わずもがな | 文艺 / 书面 |
  | 言うまでもない | 最常用 |
  | 当然だ | 普通口语 |
